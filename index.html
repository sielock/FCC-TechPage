Data Structures

Python uses many similar data types as other languages, while including a few unique ones of its own. Some of the most common are:

  • Text Type:
    • String str
  • Numeric Types:
    • Integer - int
    • Floating Point - float
    • Complex - complex
  • Sequence Types:
    • List - list
    • Tuple - tuple
  • Mapping Type:
    • Dictionary - dict
  • Boolean Types:
    • bool

Lists operate similarly to arrays in other languages, and Dictionaries are similar to associatve arrays in other languages (uses same key/value pair mapping). There are other data types, such as sets and ranges, but these are less commonly seen especially for new learners.

Flow Control

The most common forms of flow control you will encounter are if statements, for statements, and while statements.

If Statement

The if statement is used for conditional execution. It follows the if/else if pattern of other languages to evaluate multiple conditions. Python's syntax differs in that it uses elif instead of else. This if/elif sequence is used in place of a switch statement, as Python does not have the switch statment functionality other languages do.

x = int(input("Please enter an integer: ")) if x < 0: x = 0 print('Negative changed to zero') elif x == 0: print('Zero') elif x == 1: print('Single') else: print('More')

For Statement

The for statement is used to iterate over the elements of a sequence (such as a string, tuple or list) or other iterable object. Unlike most other languages, Python iterates over the items of any sequence in the order that they appear in the sequence. There is no need to set an iterator and halting condition:

words = ['cat', 'window', 'defenestrate'] for w in words: print(w)

This code will iterate through the words list, and print out each value in the order that they appear.

While Statement

The while statement is used for repeated execution as long as an expression is true:

a = 0 while a < 10: print(a) a = a + 1

This code will print the value of a, so long as the expression a < 10 remains true. Once the expression returns false, then the code will stop running. If there is an else clause present, the code within it will be executed upon the expression returning false.

After installing the latest version of Python (at the time of writing, this is Python 3.9), open your Command Prompt (Windows) or Terminal (Unix) and type:
python3.9 This will start the interpreter in interactive mode. From here, type:
